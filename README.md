# Mesq Studio

Frontend development study project focused on modern animations and motion design.

## Technologies

- **Next.js** - React Framework
- **TypeScript** - Static typing
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations and transitions
- **GSAP** - Advanced animations (if applicable)

## Installation
```bash
npm install
# or
yarn install
# or
pnpm install
```

## Run the project
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Access [http://localhost:3000](http://localhost:3000)

## Project structure
```
/app          # Pages and routes (App Router)
/components   # Reusable components
/public       # Static files
/lib          # Utilities and helpers
```

## Learning objectives

- Framer Motion animation implementation
- Scroll animations and parallax effects
- Micro-interactions and visual feedback
- Animation performance and optimization
- Responsiveness and modern design
- Component best practices

Configured for Vercel deployment.
